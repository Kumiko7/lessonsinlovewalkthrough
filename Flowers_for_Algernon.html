Walkthrough / Choices

Optional Path: The Swim Trip
There is an alternate renpy.end_replay() available via the "Swim Trip" branch. To access it:
  1. Follow the main walkthrough until you have traded with Noriko, Futaba, Chika, Sana, and Yumi.
  2. Go to the workbench and craft the Ultimate Stick Bouquet (Requires 5 Sticks AND all five character trade flags).
  3. Go to "THE SMILE MILE" and talk to Makoto; this sets buildamayakoi to True.
  4. Return home and "Use the PC". Select "Research Koi".
  5. Go to "Go somewhere" -> "UNBLEACHED BEACH". This triggers the swimtrip2 event, which ends with its own replay completion.
Phase 1: Collecting Sticks and the First Piece
1. Select "Go somewhere" → "THE SMILE MILE". This adds the first extra piece (Total: 2).
2. Answer the 10 identification questions correctly. After each correct answer, select "Yes" to continue:
    • Q1: gossamer
    • Q2: giuseppe
    • Q3: pegasus
    • Q4: halftone
    • Q5: floral laura
    • Q6: necalli
    • Q7: onund
    • Q8: harimanti
    • Q9: arramin
    • Q10: apollo
Result: You now have 10 Sticks.
Phase 2: Collecting Seeds
Travel to each location once to trigger the initial dialogue and receive seeds:
1. "Go somewhere" → "DEN OF THE MOLE RAT" (Get Poppy Seed).
2. "Go somewhere" → "CARTOGRAPHER’S CRAB-SHACK" (Get Lavender Seed).
3. "Go somewhere" → "UNBLEACHED BEACH" (Get Corpseflower Seed).
4. "Go somewhere" → "SPACY’S SUMMER BLAST..." (Get Rose Seed).
5. "Go somewhere" → "OLD(ER) DISTRICT" (Get Snapdragon Seed).
Phase 3: Crafting the Trade Items
1. Select "Use the workbench" → "Plant something".
2. Plant all five: "Plant corpseflower", "Plant lavender", "Plant poppies", "Plant roses", and "Plant snapdragons".
3. Select "Craft something" (Costs 1 stick per item):
    • Craft "Corpsecrown"
    • Craft "Rose Bouquet"
    • Craft "Lavender Perfume"
    • Craft "Snapdragon Bracelet"
    • Craft "Opium"
4. Select "Stop using the workbench".
Phase 4: Trading for the Final Pieces
Go to each location and select "Propose a trade" to give the specific item:
1. Mole Rat: Trade the "Snapdragon Bracelet".
2. Crab-Shack: Trade the "Rose Bouquet".
3. Unbleached Beach: Trade the "Corpsecrown".
4. Spacy’s: Trade the "Lavender perfume".
5. Old(er) District: Trade the "Opium".
Result: Your mayapieces count will now be 7.
Phase 5: Completion
1. Select "HOUSE" (if not already there).
2. Select "Check your Maya’s progress".
3. Select "Complete Maya assembly" to reach renpy.end_replay().
